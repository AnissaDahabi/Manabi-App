# Manabi

[![Python](https://img.shields.io/badge/Python-3.11-blue.svg)](https://www.python.org/)  [![SQLite](https://img.shields.io/badge/SQLite-3.41-blue.svg)](https://www.sqlite.org/)  [![PySide6](https://img.shields.io/badge/PySide6-6.6-blue.svg)](https://doc.qt.io/qtforpython/)

Manabi est une application de bureau pour réviser les kanjis et le vocabulaire japonais. Elle propose un système de flashcards avec répétition espacée et permet à l’utilisateur de gérer sa base de données de mots (ajout, modification, suppression, consultation).

## À propos

Manabi aide les apprenants en japonais à mémoriser efficacement les kanjis et mots de vocabulaire grâce à un **système de répétition espacée**.  
L’application utilise **SQLite** pour stocker les données et **PySide6** pour l’interface graphique.

---

## Fonctionnalités

- Affichage de tous les mots avec kanji, lecture, traduction, exemple et niveau JLPT.  
- Ajout de nouveaux mots à la base de données.  
- Modification des mots existants.  
- Suppression de mots.  
- Système de flashcards pour révisions avec répétition espacée (à revoir, difficile, correct, facile).  
- Interface graphique moderne et épurée.

---

## Technologies utilisées

- **Langage** : Python 3  
- **Base de données** : SQLite  
- **Interface graphique** : PySide6 (Qt for Python)  
- **Architecture** : Séparation vue / logique / backend
